# customer-shopping-behavior-analytics

## Project Overview

This project analyses customer shopping behaviour using Python, SQL Server, SQL, and Power BI. The objective is to transform raw retail transaction data into actionable business insights by performing data cleaning, feature engineering, database integration, SQL-based analysis, and interactive dashboard creation.

The project demonstrates an end-to-end data analytics workflow commonly used by Data Analysts and Business Intelligence professionals.

## Tech Stack

- Python (Pandas)
- SQL Server
- SQL
- Power BI
- Jupyter Notebook

## Dataset Features

The dataset contains customer shopping information including:

- Customer ID
- Age
- Gender
- Category
- Item Purchased
- Purchase Amount
- Review Rating
- Subscription Status
- Discount Applied
- Shipping Type
- Previous Purchases
- Frequency of Purchases

---

## Project Workflow

### 1. Data Cleaning & Preprocessing

Performed using Python and Pandas:

- Handled missing values in Review Ratings using category-wise median imputation.
- Standardised column names.
- Renamed columns for easier analysis.
- Removed redundant fields.
- Validated data consistency.

### 2. Feature Engineering

Created additional analytical features:

#### Age Segmentation

Customers were categorised into:

- Young Adult
- Adult
- Middle-aged
- Senior

#### Purchase Frequency Conversion

Converted textual purchase frequency values into numerical day intervals for easier analysis.

Examples:

| Frequency | Days |
|------------|------|
| Weekly | 7 |
| Fortnightly | 14 |
| Monthly | 30 |
| Quarterly | 90 |
| Annually | 365 |

---

### 3. SQL Server Integration

- Created SQL Server database.
- Connected Python with SQL Server using SQLAlchemy and PyODBC.
- Loaded transformed data into SQL Server.
- Executed advanced SQL queries for business analysis.

## Business Questions Solved Using SQL

### Revenue Analysis

- Total revenue generated by male vs female customers.

### Customer Spending Analysis

- Customers who used discounts and still spent above average.

### Product Performance

- Top 5 products with the highest average review ratings.

- Average purchase amount comparison between Standard and Express shipping.

- Average spend and total revenue comparison between subscribers and non-subscribers.

### Customer Segmentation

Customers classified as:

- New Customers
- Returning Customers
- Loyal Customers

- Top 3 purchased products within each category.

### Customer Loyalty Analysis

- Relationship between repeat purchases and subscription status.
